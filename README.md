# React Chuck Norris Jokes App

A modern React application that delivers random Chuck Norris jokes from an API. Built with TypeScript for type safety and maintainability.

## Features

- **Random Joke Generator** – Fetch fresh Chuck Norris jokes on demand
- **Keyboard Shortcuts** – Efficient navigation with intuitive shortcuts:
  - **J** – Get a new joke
  - **C** – Enter Chuck Mode
  - **F** – Add to favorites
- **Favorites Management** – Save and revisit your preferred jokes
- **Chuck Mode** – Special mode for enhanced Chuck Norris experience
- **Type-Safe** – Built with TypeScript (97.8% of codebase)
- **Responsive Design** – Optimized for desktop and mobile devices

## Technology Stack

- **React** – User interface library
- **TypeScript** – Type-safe JavaScript
- **HTML5 & CSS3** – Semantic markup and styling
- **Vite** – Fast build tooling and development server
